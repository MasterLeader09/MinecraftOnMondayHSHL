1. tracking.py
    --> Schreibe ein Programm welches egal wo dein Avatar hinläuft einen Goldblock an die Füße deines Avatars setzt
    --> Der Goldblock soll allerdings wieder verschwinden, wenn der Avatar nicht mehr auf ihm steht

2. suchdenort.py
    --> Hier sollst du ein Spiel erstellen
    --> Das Spiel soll einen zufällige Position in der Minecraftwelt auswählen
    --> Nach einem kleinen Countdown hat der Spieler eine Minute Zeit, um die Position zu finden
    --> Findet der Spieler die Position, so soll ihm die benötigte Zeit angezeigt werden
    --> Findet der Spieler nicht die Position, so soll ihm eine Nachricht über das Scheitern in Minecraft ausgegeben werden
    --> Als Hilfestellung für den Spieler darf nur die absolute Entfernung vom Spieler zur Position im Spiel ausgegeben werden (nicht die Abstände auf den einzelnen Achsen)
    --> In diesem Ordner soll ein Textdoktument angelegt werden, in welchem die Statistik jedes einzelnen Spiels festgehalten wird
    --> Zu speichern sind folgende Daten (Erfolg/Scheitern, benötigte Zeit, Datum, Uhrzeit(des Spielbeginns))
    --> Wenn ein Spiel vorüber ist, soll auch immer die Statistik des besten Versuchs ausgegeben werden
