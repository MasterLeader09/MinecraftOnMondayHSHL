Dieses Repository entstammt von Nils Reuter (wissenschaftlicher Mitarbeiter der Hochschule Hamm-Lippstadt) und sind Bestandteil der Workshopreihe "Spielerischer Einstieg in Python" des zdi-Schülerlabors der Hochschule Hamm-Lippstadt.

Das komplette Kursangebot des zdi-Schülerlabors finden Sie unter dem folgenden Link https://www.hshl.de/junior-campus/zdi-schuelerlabor/ (Das komplette Angebot ist kostenfrei).

Die hier erstellten Dateien wurden zum Lernen einer Programmiersprache (in diesem Fall Python3) erstellt und stehen jedem Menschen zur nicht kommerziellen Nutzung zur Verfügung).

Die Programme wurden für das Spiel Minecraft Pi Edition erstellt. Das Spiel ist Bestandteil des Betriebssystems Raspbian (Noobs), welches kostenfrei unter dem Link https://www.raspberrypi.org/downloads/ erhältlich ist.
